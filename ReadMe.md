# Система напарников

*   Добавлена система напарников.
    

*   Обратившись к NPC Люмину в Верхнем Дозоре после достижения 66 уровня, можно выполнить сюжетное задание "Яйцо духов" и получить напарника Коко-минмин.
    

*   Добавлено новое окно интерфейса “Список помощников”.
    

*   В нём находятся все зарегистрированные питомцы и напарники персонажа.
    

*   “Список помощников” можно найти в главном меню “Социальное”.
    

*   Минимальное количество слотов для помощников - 10, максимальное - 100.
    

*   Прежние руководства призыва питомцев изменены на "Сертификаты", призыв возможен после регистрации питомца.
    
    *   Ранее изученные руководства призыва будут удалены из активных умений и отправлены в виде сертификатов на почту персонажа.
        
    *   Содержимое сумки питомцев будет сохранено.
        
*   Ранее, нельзя было добавлять больше одного питомца одного вида.
    

*   Теперь можно добавить одинаковых питомцев и развивать их по отдельности.
    

## Развитие питомцев

*   Добавлены уровни питомцев, всего 10 ступеней.
    

*   Уровень можно повысить с помощью разных предметов.
    

*   Материалы для зачарования
    
*   Материалы для сбора и ремесла
    
*   Снаряжение для синтеза
    
*   Реликвии
    
*   Кристаллы
    
*   Конфеты для питомцев
    

*   После достижения 10 уровня, с помощью камней эволюции и золота, питомца можно эволюционировать в напарника.
    

*   Камни эволюции можно создать с помощью ремесла “Алхимия”.
    

*   Чертежи для камней эволюции можно купить у Торговца товарами для алхимии.
    

*   Питомцев можно синтезировать.
    

*   С помощью синтеза двух питомцев, можно получить случайного питомца или напарника.
    
*   Чем выше уровень двух питомцев, тем выше вероятность создать напарника.
    

*   Добавлен новый способ получения питомцев.
    

*   Собрав "Кристаллы мира" с обычных монстров и боссов выше 65 уровня, их можно обменять на сертификаты питомцев.
    

*   Добавлены новые угощения для питомцев.
    

*   Существуют обычные, качественные, и необычное угощения.
    
*   Можно создать с помощью ремесла “Готовка” или купить у торговцев обычными товарами.
    

*   Рецепты угощений можно купить у торговцев снастями для рыбалки.
    

## Структура напарников

*   Ранг напарника можно повысить путем продвижения.
    

*   Максимальный ранг напарника - 10.
    
*   Продвинуть напарника можно только с помощью другого напарника.
    

*   При повышении ранга уровень симпатии повышается на случайное значение.
    

*   С повышением уровня симпатии повышаются свойства умений поддержки.
    
*   Максимальный уровень симпатии - 50.
    

*   Изначально, напарники не обладают способностями таких типов как “авто-сбор”, “авто-подбор”.
    

*   С помощью функции “Перенос” можно перенести способности любого питомца напарнику.
    

*   Вместо прежней энергии питомцев, напарникам добавлен параметр усталости.
    

*   После призыва усталость напарника увеличивается на 1 ед. каждую минуту.
    
*   Бодрость напарника можно увеличить с помощью подарков.
    

*   Подарки можно изготовить или купить у торговца.
    

*   Чертеж предмета можно купить у торговца товарами для обработки.
    

## Умения напарников

*   Умение поддержки
    
    *   Доступно каждые 3 мин. после призыва напарника.
        
    *   Временно повышают боевые способности персонажа.
        

*   Таланты
    
    *   При добавлении напарника он получает 1 талант.
        
        *   Каждый талант имеет три уровня.

**Талант**

**Эффект**

Близкий контакт

Небольшой шанс увеличить время действия умений поддержки во время сражения.

Первая помощь

При уменьшении HP в течение определенного времени с низкой вероятностью ежесекундно восстанавливает определенное количество HP.

Подмога

При уменьшении MP в течение определенного времени с низкой вероятностью ежесекундно восстанавливает определенное количество MP.

Щит

При уменьшении HP в течение определенного времени с низкой вероятностью поглощает определенное количество урона.

Ускорение роста

Если дать подарок напарнику, в течение определенного времени с низкой вероятностью увеличивает количество опыта, получаемого во время охоты.

Ускорение чар

Если дать подарок напарнику, в течение определенного времени с низкой вероятностью увеличивает количество опыта умений, получаемого во время охоты.

Мгновенное ускорение

Если дать подарок напарнику, в течение определенного времени с низкой вероятностью увеличивает количество опыта персонажа и опыта умений, получаемое во время охоты.

Подмастерье

Во время занятия ремеслом повышается вероятность создать шедевр.

Рыбак

Во время рыбалки повышается вероятность поймать крупную рыбу.

Собиратель

С низкой вероятностью уменьшает время сбора.

# Новое подземелье

## Гнездо Паркин

*   Подземелье предназначено для группы из 5 игроков 66 уровня.
    
*   Подбор группы в подземелье доступен при уровне снаряжения не ниже 453.
    
*   Вход в подземелье находится в Эссении > Эссенийский хребет.
    
*   Добавлено новое сюжетное задание «Рождение бабочек».
    

*   Можно выполнить после выполнения задания 66 уровня «Между падением и приземлением».
    

*   Награды за прохождение подземелья:
    

*   Ящик со снаряжением ревущей ярости
    
*   Кристалл мира
    
*   Опыт умений
    
*   Необычный металл
    
*   Усилитель украшений I
    
*   Мистическая вода богов
    
*   Старые припасы
    
*   Коготь черта
    
*   Череп дракона
    
*   Драконья основа для лука
    
*   Последний элемент
    
*   Маска крови Эллеона
    
*   Бриллиант
    
*   Маска-невидимка и прозрачные украшения для спины.
    

# Баланс классов

*   Добавлены символы для умений пробуждения.
    

*   Коробка с пробужденным символом дается в награду за выполнение достижения "Новые силы".
    
*   Чтобы выполнить его, необходимо пробудить персонажа и получить достижение "Внутреннее спокойствие", а также развить персонажа до 70-го уровня и получить достижение "Вот и 70!".
    

## Воин

*   Нарастающая ярость
    

*   Эффект повышения скорости применения таланта в связке с “Нарастающей яростью” действует для 1 и 2 ударов.
    
*   Уменьшена максимальная скорость применения таланта “Нарастающая ярость” с 50% до 25%.
    

## Берсерк

*   Урон Смертельного удара изменен в зависимости от уровня HP.
    

*   Коэффициент изменения силы увеличен для промежутка от 50 до 100% НР.
    
*   Коэффициент изменения силы уменьшен для уровня HP ниже 50%.
    

*   При использовании блока топором расходуется 50 MP.
    

## Маг

*   Слияние
    

*   Скорость применения умений "Буйство стихий", "Кристаллическое касание", "Энергетический шторм", "Конвергенция элементов" повышен на 10%.
    
*   Скорость снарядов для умений "Буйство стихий" и "Кристаллическое касание" повышен на 10%.
    

## Жрец

*   Крайняя мера
    

*   Увеличено время, в течение которого  можно использовать комбо после использования умений "Тройное возмездие" и "Вихрь Зенобии".
    

*   Заклинание тела и души
    

*   Восстановление 3%-8% MP в секунду в зависимости от уровня заряда изменено на 232-774 ед.
    

## Мистик

*   Удар метаморфозы
    

*   Увеличено время, в течение которого можно использовать умение быстрее после использования "Взрыва метаморфозы".
    

*   Сфера жизни Аруна
    

*   Время призванной сферы увеличено до 180 сек.
    

*   Слезы Аруна
    

*   Время призванной сферы увеличено до 180 сек.
    
*   Восстановление 20% MP изменено на восстановление 1546 ед.
    

*   Миньон-целитель
    

*   При использовании Миньона-целителя в состоянии "Вселение духа" восстановление 10%  MP изменено на восстановление 773 ед.
    

## Крушитель

*   Яростный удар сверху
    

*   Скорость применения умения повышена на 10%.
    
*   Дальность перемещения уменьшена с 20 до 8 м.
    

## Рыцарь

*   Гарпун
    

*   Увеличен радиус захвата цели.
